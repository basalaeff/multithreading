# Лабораторная работа №4: Многопоточность (multithreading)

## Описание
Проект является реализацией лабораторной работы 4 по теме «Многопоточность» курса «Программирование».
*Цель работы* — изучение принципов реализации многопоточности и практические применение возможностей языков
высокого уровня при моделировании бизнес-процессов.
*Задание к работе:* Самостоятельно решить задачи в соответствии с индивидуальным вариантом 4.

## Установка и сборка
```bash
git clone https://github.com/basalaeff/multithreading.git
```
```bash
cmake -S . -B build -G Ninja
cmake --build build
```

## Задание
*1. Реализация абстрактных структур данных. Реализовать параллельный запуск заданного числа потоков, содержащих случайные символы из ASCII таблицы в формате гонки.*
Проанализировать особенности (сравнительный анализ):
- Mutexes
- Semaphore
- Barrier
- SpinLock
- SpinWait
- Monitor.

Проанализировать скорости работы примитивов с использованием StopWatch и BenchMarkDotNet.

*2. Многопоточная обработка данных.*
Задаются данные: размер массива данных, количество параллельных потоков, значения атрибутов структуры, дополнительные данные (в зависимости от варианта).
Результатом работы программы являются вывод времени обработки без использования многопоточности, времени обработки с использованием многопоточности и результатов обработки.
4. Структура содержит данные о доставленных посылках (код товара, населенный пункт, ФИО получателя). Необходимо по шаблону кода товара Ш определить фамилии получателей. Шаблон – регулярное выражение.
